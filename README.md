# AI Job Recommendation System

AI-based Job Recommendation System using NLP, TF-IDF, KNN, Random Forest and ANN.

## Features
- Job Recommendation
- Resume Upload
- Skill Extraction
- KNN Model
- Random Forest Model
- ANN Model
- Match Percentage

## Technologies
- Python
- Flask
- TensorFlow
- Scikit-Learn
- Pandas
- NumPy

## Accuracy
- KNN: 78.65%
- Random Forest: 80.72%
- ANN: 84.32%
